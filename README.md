# AI Fake News Detector

AI Fake News Detector developed using Machine Learning and Flask.

## Features

- Fake News Detection
- Real News Detection
- 99.31% Accuracy
- TF-IDF Vectorization
- Passive Aggressive Classifier
- Flask Web Application

## Technologies

- Python
- Flask
- Scikit-Learn
- Pandas
- Joblib

## Dataset

Fake and Real News Dataset
